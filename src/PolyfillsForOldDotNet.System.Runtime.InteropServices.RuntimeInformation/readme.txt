   ===========
   DESCRIPTION
   ===========
   Contains polyfills for the `OSArchitecture` property and `IsOSPlatform` method
   of `RuntimeInformation` class from
   the System.Runtime.InteropServices.RuntimeInformation library.

   =============
   RELEASE NOTES
   =============
   Performed a migration to the modern C# null/not-null checks.
